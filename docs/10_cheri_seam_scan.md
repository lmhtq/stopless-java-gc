# CHERI seam scan — HotSpot source

Generated by `scripts/scan_cheri_seams.py`.

Each entry is a *candidate* CHERI seam: a site where HotSpot
conflates a 'machine word' with 8 bytes, or where an integer-
derived pointer might lose its CHERI capability tag. Many entries
are harmless on standard LP64 but break (or already broke) under
purecap. Use this as a *worklist* — verify each before patching.
